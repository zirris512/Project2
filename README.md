# Epic Adventure!

Epic Adventure is a choose your own adventure story. The user makes choices and winds up at an ending based on those choices.

## Technologies

* HTML/CSS
* Javascript
* JQuery
* Pug
* MySQL
* Sequelize
* Express
* Node.js
